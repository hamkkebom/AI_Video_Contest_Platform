Product Requirements Document (PRD)

버전: v0.1-mockup | 상태: 목업 기준 초안 | 수정일: 2026-02-19

1. 기능 요구사항

함께봄 AI 영상 공모전 플랫폼의 Phase 1 핵심 기능. 각 기능은 고유 ID(F-001~)로 식별됨.

기능 ID 기능명 설명 대상 역할
F-001 공모전 접수 참가자가 공모전을 탐색하고 영상을 제출 participant
F-002 공모전 심사 심사위원이 제출된 영상을 평가하고 점수 부여 judge
F-003 결과 발표 공모전 종료 후 순위 및 수상자 공개 all
F-004 영상 갤러리 제출된 영상들을 갤러리 형식으로 전시 guest, participant
F-005 작품 좋아요 갤러리에서 영상에 좋아요 기능 guest, participant
F-006 심사자 초대 공모전 주최자가 심사위원 초대 host
F-007 역할 전환 사용자 역할 변경 (guest → participant, participant → host) all
F-008 UTM 추적 마케팅 채널별 사용자 유입 추적 admin
F-009 심사 자동화 AI 기반 심사 자동화 (Phase 2) host, judge
F-010 대행 연결 제작 대행 플랫폼 연결 (Phase 2) participant
F-011 갤러리 필터 카테고리/태그 기반 갤러리 필터링 guest, participant

2. 비기능 요구사항

분류 요구사항 목표값
성능 페이지 로드 시간 < 2초 (LCP)
성능 API 응답시간 < 500ms (평균)
보안 XSS 취약점 제거 OWASP Top 10 준수
보안 CSRF 토큰 모든 POST 요청에 적용
확장성 동시 사용자 1,000명 이상
접근성 WCAG 2.1 AA 레벨 준수

3. 우선순위 매트릭스 (MoSCoW)

우선순위 기능 설명
Must F-001, F-002, F-003, F-004 공모전 핵심 플로우 필수
Should F-005, F-006, F-007, F-008 사용자 경험 및 관리 기능
Could F-009, F-010, F-011 Phase 2 이후 확장 기능

4. 제약사항

제약사항 설명 영향도
기술 스택 고정 Next.js 15, Supabase, Mux (변경 불가) High
외부 의존성 Supabase Auth, Mux API (가용성 의존) High
영상 용량 최대 1GB (Mux 제한) Medium
지연 로딩 ISR 기반 캐싱 (최대 3600초) Medium
