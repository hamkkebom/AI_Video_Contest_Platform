함께봄 AI 영상 공모전 플랫폼 - 제품 요구사항 정의서

문서 정보: v0.1-mockup | 최종 수정: 2026-02-19 | 상태: 목업 기준 초안

기능요구사항

플랫폼의 핵심 기능 목록입니다. 각 기능은 고유한 ID(F-001 ~ F-011)로 식별됩니다.

기능 ID 기능명 설명 우선순위
F-001 공모전 접수 참가자가 공모전에 영상 작품을 제출하는 기능 P0
F-002 공모전 심사 심사자가 제출된 작품을 평가하고 점수를 부여하는 기능 P0
F-003 결과 발표 공모전 심사 결과를 공개하고 수상작을 발표하는 기능 P0
F-004 영상 갤러리 제출된 영상 작품을 갤러리 형식으로 전시하는 기능 P1
F-005 작품 좋아요 사용자가 마음에 드는 작품에 좋아요를 표시하는 기능 P1
F-006 심사자 초대 공모전 주최자가 심사자를 초대하고 권한을 부여하는 기능 P0
F-007 역할 전환 사용자가 참가자/심사자/주최자 역할을 전환하는 기능 P1
F-008 UTM 추적 마케팅 캠페인 성과를 추적하기 위한 UTM 파라미터 지원 P2
F-009 심사 자동화 AI를 활용한 자동 심사 및 점수 산출 기능 P1
F-010 대행 연결 외부 대행사와의 연동 및 통합 기능 P2
F-011 갤러리 필터 갤러리에서 카테고리, 기간, 평점 등으로 작품을 필터링하는 기능 P1

비기능요구사항

시스템의 성능, 보안, 확장성 등 비기능적 요구사항입니다.

카테고리 요구사항 목표값
성능 페이지 로딩 시간 < 2초
성능 동시 사용자 지원 10,000명
가용성 시스템 가용성 99.9%
보안 데이터 암호화 TLS 1.2 이상
확장성 데이터베이스 확장 수평 확장 지원
호환성 브라우저 지원 Chrome, Firefox, Safari, Edge 최신 2버전

우선순위매트릭스

기능의 중요도와 긴급도를 기반으로 한 우선순위 분류입니다.

우선순위 분류 기능 ID 설명
P0 필수 (Critical) F-001, F-002, F-003, F-006 공모전 운영의 핵심 기능. 즉시 구현 필요
P1 높음 (High) F-004, F-005, F-007, F-009, F-011 사용자 경험 향상 기능. 1차 릴리스에 포함
P2 중간 (Medium) F-008, F-010 부가 기능. 2차 이후 릴리스에 포함

제약사항

프로젝트 진행 시 고려해야 할 기술적, 비즈니스적 제약사항입니다.

제약사항 내용 영향도
기술 스택 React + Node.js + PostgreSQL 기반 구축 높음
개발 기간 6개월 내 1차 릴리스 완료 높음
예산 인프라 비용 월 $5,000 이내 중간
규정 준수 개인정보보호법(PIPA) 준수 필수 높음
영상 용량 최대 파일 크기 2GB 제한 중간
동시 심사 최대 100명의 심사자 동시 접속 지원 중간
